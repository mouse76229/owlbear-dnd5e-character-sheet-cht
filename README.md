# D&D 5E Character Sheet (Traditional Chinese / 繁體中文版)

![Owlbear Rodeo Compatible](https://img.shields.io/badge/Owlbear%20Rodeo-Compatible-purple)
![D&D 5E](https://img.shields.io/badge/D%26D-5th%20Edition-red)

**[English]**
This project is a **D&D 5th Edition character sheet** for Owlbear Rodeo, adapted from the original [Owlbear ShadowDark Character Sheet](https://github.com/maxpaulus43/owlbear-shadowdark-character-sheet) created by Max Paulus.
It has been fully converted to support D&D 5E rules and localized in Traditional Chinese (Taiwan).

**[中文]**
本專案為適用於 Owlbear Rodeo 的 **D&D 第五版角色卡**，改編自 Max Paulus 製作之 [Owlbear ShadowDark Character Sheet](https://github.com/maxpaulus43/owlbear-shadowdark-character-sheet)。
本版本已完整轉換為 D&D 5E 規則系統，並全面繁體中文化（台灣用語）。

---

## Installation / 安裝

**[English]**
To use this extension in Owlbear Rodeo:

1.  Copy the Manifest URL below:
    ```
    https://mouse76229.github.io/owlbear-dnd5e-character-sheet-cht/manifest.json
    ```
2.  Open a room in [Owlbear Rodeo](https://www.owlbear.rodeo/).
3.  Click the **Extras** (three dots) menu in the bottom-left toolbar.
4.  Select **Extensions**.
5.  Click the **+** (plus) button to add a custom extension.
6.  Paste the URL and click **Install**.

**[中文]**
若要在 Owlbear Rodeo 中使用此擴充功能：

1.  複製下方的 Manifest 網址：
    ```
    https://mouse76229.github.io/owlbear-dnd5e-character-sheet-cht/manifest.json
    ```
2.  開啟 [Owlbear Rodeo](https://www.owlbear.rodeo/) 房間。
3.  點擊左下角工具列的 **Extras**（三個點圖示）選單。
4.  選擇 **Extensions**（擴充套件）。
5.  點擊 **+**（加號）按鈕以新增自訂擴充套件。
6.  貼上網址並點擊 **Install**（安裝）。

## Features / 功能

**[English]**
*   **D&D 5E Rules**: Full support for 5th Edition mechanics including:
    *   Proficiency bonus based on level
    *   18 Skills with proficiency tracking
    *   6 Saving throws
    *   AC calculation (Light/Medium/Heavy armor + Shield)
    *   Experience points and level-up system (Levels 1-20)
    *   Death saving throws
    *   Conditions tracking
*   **Fully Localized**: UI, weapons, armor, spells, and descriptions in Traditional Chinese.
*   **Save Slots**: Save up to 5 different characters.
*   **Import/Export**: Export and import character data in JSON format.
*   **Automation**: Auto-calculate AC, ability modifiers, attack bonuses, and spell save DC.
*   **Dice Rolling**: Integrated dice rolling with notifications.
*   **Undo/Redo**: Full history tracking for character changes.

**[中文]**
*   **D&D 5E 規則**：完整支援第五版機制，包括：
    *   依等級計算熟練加值
    *   18 種技能與熟練度追蹤
    *   6 種屬性豁免檢定
    *   AC 計算（輕/中/重甲 + 盾牌）
    *   經驗值與升級系統（1-20 級）
    *   死亡豁免檢定
    *   異常狀態追蹤
*   **全面中文化**：介面、武器、防具、法術與描述皆已翻譯為繁體中文。
*   **存檔欄位**：提供 5 個角色存檔欄位。
*   **匯入/匯出**：支援 JSON 格式的角色資料匯出與匯入。
*   **自動計算**：自動計算 AC、屬性修正值、攻擊加值與法術豁免 DC。
*   **擲骰功能**：內建擲骰按鈕與結果通知。
*   **復原/重做**：完整的角色變更歷史記錄。
